Achsen Bearbeitungsmodus wechseln

Unter dem Menüpunkt Achsen Bearbeitungsmodus wechseln (im Menü Administration) können Sie als Applikationsadministrator für jede Organisation den Bearbeitungsmodus der Achsen einstellen (siehe Abbildung 70).

Abbildung 70: Achsen Bearbeitungsmodus wechseln

Warnung: Der Wechsel des Achsen Bearbeitungsmodus kann nicht rückgängig gemacht werden. Führen Sie die Änderung nur durch, wenn der Mandant dies ausdrücklich wünscht.

 

Tipp: Mittels Jahresabschluss rückgängig machen kann der Benutzeradministrator einen Jahresabschluss zurücksetzen. Damit kann auch ein versehtlicher Wechsel zum Achsen Bearbeitungsmodus rückgängig gemacht werden.
