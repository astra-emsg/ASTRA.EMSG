EMSG Hilfe
Check-Out von Inspektionsrouten rückgängig machen

Unter dem Menüpunkt Check-Out von Inspektionsrouten rückgängig machen (im Menü Administration) können Sie als Benutzeradministrator die exportierte Inspektionsrouten mittels  erneut für die Bearbeitung im EMSG Master freigeben.

Warnung: Diese Funktionalität soll nur verwendet werden, wenn der Import der Inspektionsroute in den Master nicht mehr möglich ist. Wurde der Check-Out rückgängig gemacht, ist der Import der Inspektionsroute nicht mehr möglich! Die Inspektionsroute müsste erneut exportiert werden. Die Zustandserfassung muss erneut durchgeführt werden.
