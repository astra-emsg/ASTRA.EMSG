EMSG Hilfe
Startseite

Um dem Benutzer eine übersichtliche Bedienung zu ermöglichen bietet die EMSG Startseite eine Vielzahl von Bedienelemente an (siehe Abbildung 2).

 

Abbildung 2: Startseite

Netz und Zustand erfassen: Unter diesen Abschnitt findet der Benutzer Funktionen zum Verwalten des Strassennetzes und der Zustandserfassung.

Massnahmen verwalten: Hier werden Verwaltungsfunktionen für Massnahmen aufgelistet.

Auswertungen durchführen: Die wichtigsten Auswertungen für den ausgewählten Arbeitsmodus findet der Benutzer unter diesem Abschnitt.

Systeminformation: Hier werden Informationen über die Benutzerrolle und den Arbeitsmodus angezeigt.  Ausserdem kann der Benutzer den Mandanten wechseln.

Breadcrumb: Systemweit steht die Navigation via Breadcrumb zur Verfügung.

Help Desk: Über diesen Link kann der Benutzer die Web-Seite des MISTRA-Help Desks öffnen.

Menüleiste: Der Benutzer kann systemweit über die Menüleiste navigieren. Diese stellt sämtliche für die Benutzerrolle möglichen Funktionen zur Verfügung (auf der Startseite werden nur die wichtigsten bereitgestellt).

Sprachauswahl: Über die Sprachauswahl kann der Benutzer zwischen Deutsch, Französisch und Italienisch umschalten.

Hilfe: Über diese Schaltfläche gelangt der Benutzer zur EMSG-Hilfeseite.

Achtung: Der Benutzer hat nicht bei jedem Arbeitsmodus Zugriff auf alle Funktionen. Möchte er eine Funktion nutzen, welche auf der Startseite ausgegraut ist, muss er den Arbeitsmodus wechseln.
