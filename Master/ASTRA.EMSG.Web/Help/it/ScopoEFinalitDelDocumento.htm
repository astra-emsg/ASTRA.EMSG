EMSG Hilfe
Scopo e finalità del documento

Il manuale d'uso fornisce tutte le informazioni necessarie all'utente del sistema EMSG per gestire il sistema secondo le regole e per poter reagire correttamente in caso di problemi.

La presente versione del manuale d'uso tratta dei casi di applicazione implementati nella release 1. Il manuale d'uso viene ampliato in caso di eventuali ampliamenti di EMSG-Master e rispecchia così la versione attuale di EMSG-Master.

Questo manuale d'uso contiene le funzionalità dell'applicazione web "EMSG-Master". L'applicazione Windows "EMSG-Mobile" per il rilevamento mobile di dati viene descritta in [3].
