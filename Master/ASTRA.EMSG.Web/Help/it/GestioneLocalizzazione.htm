EMSG Hilfe
Gestione localizzazione

EMSG supporta la lingua tedesca, francese e italiana. L'adattamento dell'interfaccia utente di EMSG avviene tramite i cosiddetti file di risorse in formato XML. Alla voce del menu Gestione localizzazione (nel menu Amministrazione), in qualità di amministratore dell'applicazione è possibile scaricare questi file di risorse oppure modificarli e caricarli di nuovo.

Per l'adattamento dell'interfaccia utente procedere come segue:

1.     Fare clic sul pulsante Scarica i file di localizzazione attuali per scaricare dal server i file di risorse in formato ZIP.

2.     Salvare il file ZIP in locale e decomprimerlo.

3.     Individuare il file necessario. I nomi dei file sono strutturati come segue: <Contenuto>Localization.<Lingua>.resx, dove <Contenuto> indica il tipo di testi contenuti nel file (es. "Menu" per tutte le voci del menu) e <Lingua> è la sigla della lingua (de, fr, it). Un elenco dei file di risorse realmente presenti è disponibile nella
Tabella 10.

4.     Aprire il file di risorse con l'editor.

Suggerimento: per piccole modifiche (es. la correzione di errori ortografici) è sufficiente un editor di base (es. Notepad). Per ampie modifiche si consiglia Microsoft Visual Studio oppure uno strumento specifico per file di risorse (es. ResX Localization Studio).

 

5.     Cercare il punto preciso del file utilizzando la funzione di ricerca dell'editor per trovare la stringa di caratteri che si desidera modificare.

6.     Apportare tutte le modifiche desiderate e salvare il file.

7.     Fare clic sul pulsante Seleziona... e caricare il file di risorse modificato. È possibile caricare solo un file di risorse alla volta.

Avviso: eseguire modifiche ai file di risorse solo quando è richiesta la finestra di manutenzione poiché se si carica un file di risorse nel momento in cui l'utente sta lavorando sull'applicazione è possibile perdere i dati.

 

Indicazione: non bisogna dimenticare che le modifiche all'interfaccia utente devono essere riportate in modo adeguato anche alla documentazione, in particolare nel manuale d'uso (ovvero il manuale d'uso per EMSG-Mobile [3]) e nella guida in linea (vedere sezione 7.11).
