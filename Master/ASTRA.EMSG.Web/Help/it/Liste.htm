Liste

Per fornire una panoramica all'utente vengono create alcune liste che vengono visualizzate come segue (vedere Figura 3).

Figura 3: liste in EMSG

Le liste consentono le seguenti funzioni di base:

 Un clic sul pulsante  Rileva nuovo gruppo di dati (la dicitura cambia in base alla lista) fa aprire una finestra di dialogo. In questo modo il Data Manager può creare un nuovo gruppo di dati.

 Un clic sul pulsante  Modifica fa aprire la stessa finestra di dialogo per la successiva modifica di un gruppo di dati presente.

 Un clic sul pulsante  Elimina rimuove un gruppo di dati presente (dopo una domanda di sicurezza).

 Nel caso in cui la lista non sia su un'unica pagina, tramite le funzioni di scorrimento (prima pagina, pagina precedente, pagina successiva, ultima pagina) è possibile passare alle altre pagine. Inoltre, in basso a sinistra vi è anche un pulsante per il ricaricamento dei dati (nel caso in cui questi siano stati cambiati da un altro utente nel frattempo).

Suggerimento: in lunghe liste la ricerca tramite le funzioni di scorrimento può risultare un po' difficoltosa in presenza di molte pagine. Utilizzare le funzioni di filtro (diverse a seconda della lista) per diminuire il numero di pagine. Nei filtri di testo è possibile utilizzare il simbolo "_" (trattino basso) per filtrare i caratteri desiderati.
