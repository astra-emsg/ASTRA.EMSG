EMSG Hilfe
Struttura del sistema e interfacce esterne

L'applicazione tecnica EMSG è costituita dalle due applicazioni seguenti:

1.     EMSG-Master: l'applicazione principale, un'applicazione web con piene funzionalità per un utilizzo sul posto di lavoro. Questa applicazione tecnica è descritta nel presente documento.

2.     EMSG-Mobile: un'applicazione Windows con funzioni limitate che può essere utilizzata su terminali mobili (tablet PC). Viene descritta in [3].

Lo scambio di dati tra l'EMSG-Master e l'EMSG-Mobile funziona "offline", cioè avviene manualmente tramite un supporto dati (es. una chiavetta USB).

Vengono scambiati i seguenti dati:

·         Da EMSG-Master a EMSG-Mobile: itinerari d'ispezione con il materiale cartografico corrispondente in modo tale che l'EMSG-Mobile possa essere utilizzato senza collegamento internet. La relativa funzione utente è descritta alla sezione 4.3.3.

·         Da EMSG-Mobile a EMSG-Master: i dati di stato registrati per un itinerario di ispezione e interventi consigliati. La relativa funzione utente è descritta anche alla sezione 4.3.3.

Inoltre, l'EMSG-Master dispone delle seguenti interfacce online ad altri sistemi:

·         Sistema di base MISTRA: tramite il sistema di base, l'EMSG riceve i dati relativi agli assi per mezzo di un'interfaccia standardizzata (servizio web e formato file INTERLIS 2). La funzione utente per l'aggiornamento dei dati degli assi è descritto alla sezione 7.5. Nel sistema di base non vengono riprodotti dati tecnici.

·         LDAP: la gestione dei ruoli utente avviene all'interno del LDAP di MISTRA. EMSG-Master legge queste informazioni dal LDAP per autorizzare gli utenti. Per ogni utente autorizzato in EMSG-Master, i suoi ruoli EMSG vengono letti dal servizio web LDAP. Per l'utente significa accedere automaticamente a EMSG con il proprio login MISTRA. La gestione dei ruoli utente avviene all'interno di MISTRA.

·         swisstopo: consente di impostare le immagini di sfondo per il modo SIG dell'EMSG. Possono essere ortofoto, materiale cartografico o dati MU. Informazioni sulla visualizzazione di immagini di sfondo sono disponibili alla sezione 4.3.
