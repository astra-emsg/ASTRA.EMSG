Finestre di dialogo

La modifica dei dati avviene generalmente in finestre di dialogo modali, ciò significa che è necessario chiudere la finestra di dialogo prima di poter utilizzare altre funzioni.

Vi sono generalmente quattro possibilità:

1.          Il sistema salva i dati inseriti. La finestra di dialogo rimane aperta ed eventuali valori calcolati vengono aggiornati.

2.          Il sistema salva i dati inseriti e chiude la finestra di dialogo.

3.          Il sistema annulla la modifica. La finestra di dialogo viene chiusa senza salvare i dati inseriti.

4.          Il sistema elimina il gruppo di dati (dopo una domanda di sicurezza).

Avviso: i gruppi di dati vengono eliminati fisicamente. Ciò significa che non c'è nessun "cestino", dal quale è possibile recuperare i gruppi di dati cancellati per sbaglio. Si perdono anche i gruppi di dati dipendenti (es. informazioni di stato relative a un tratto di strada).
