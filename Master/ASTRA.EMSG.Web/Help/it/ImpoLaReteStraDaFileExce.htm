Importare la rete stradale da file Excel

Scorrere la lista per la gestione dei tratti di strada (vedere sezione 4.2.1).

Un clic sul pulsante Importa rete stradale (.XLSX) (vedere Figura 11) fa aprire una finestra di dialogo per l'importazione della rete stradale da un file Microsoft Excel.

Indicazione: è supportato solo il formato XLSX (Excel 2007, 2010 o 2013). Se si utilizza una versione più vecchia di Excel (Excel 2000, XP o 2003), è necessario installare il "Microsoft Office Compatibility Pack".

 

Figura 14: finestra di dialogo "Importa rete stradale" (scelta del file)

 Un clic sul link Scarica il template fa aprire un file Excel nel quale è presente una guida alla compilazione per l'importazione. È possibile compilare il file e salvarlo localmente.

 Un clic sul pulsante Seleziona... fa aprire una finestra di dialogo standard per l'apertura di un file. È possibile scegliere il file Excel salvato localmente che si desidera importare.

 Un clic sul pulsante Carica file avvia il controllo del file Excel. Se la struttura del file non coincide con il Template oppure se si verifica un errore di validazione, viene emesso un avviso e non viene importato alcun dato.

Con la funzione di importazione è possibile sia inserire nuovi tratti di strada sia aggiornare tratti esistenti. Se i campi Nome della strada, Denominazione da e Denominazione a in una cella Excel coincidono con un tratto di strada presente, allora quest'ultimo viene aggiornato con le informazioni del file Excel (cioè le informazioni del file Excel sovrascrivono le informazioni in EMSG), in caso contrario viene creato un nuovo tratto di strada.

Dopo un controllo con esito positivo compare una panoramica dei nuovi tratti di strada inseriti e di quelli aggiornati (vedere Figura 15). Verificare con cura se il risultato corrisponde alla propria idea e solo in seguito fare clic in fondo alla panoramica sul pulsante Esegui import. In seguito l'import non potrà più essere annullato.

Figura 15: finestra di dialogo "Importa rete stradale" (controllo)

Avviso: se ad esempio tra l'upload del file Excel e l'avvio dell'importazione, l'utente, la denominazione da o la denominazione a dei tratti di strada interessati cambiano, la procedura di importazione non darà il risultato che si può vedere nell'anteprima. Garantire sul lato organizzativo che ciò non possa accadere.
