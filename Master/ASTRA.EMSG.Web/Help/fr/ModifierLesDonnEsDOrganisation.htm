Modifier les données d'organisation

Le point de menu Modifier les données d'organisation (dans le menu Administration) vous permet, en tant qu'administrateur des utilisateurs, de gérer les valeurs caractéristiques de votre commune sur lesquelles le benchmarking devra s'appuyer, afin que vous puissiez vous comparer avec les autres communes sur la base de valeurs caractéristiques semblables.
