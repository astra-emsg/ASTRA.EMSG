EMSG Hilfe
Page de démarrage

Pour que l'utilisateur bénéficie d'une expérience d'utilisation aussi structurée que possible, la page de démarrage d'EMSG offre une multiplicité d'éléments d'exploitation (voir Figure 2).

 

Figure 2: Page de démarrage

Gérer le réseau et l'état: dans cette rubrique, l'utilisateur trouvera les fonctions de gestion du réseau routier et de l'état des routes.

Gérer les interventions: cette rubrique donne la liste des fonctions de gestion des interventions.

Exécuter les exploitations: l'utilisateur trouvera dans cette rubrique les principales exploitations pour le mode de travail sélectionné.

Information système: cette rubrique fournit des informations sur les droits d'utilisateur et le mode de travail.  L'utilisateur a également la possibilité de changer le mandant.

Fil d'Ariane: aide à la navigation dans l'ensemble du système.

Help Desk: ce lien permet à l'utilisateur d'ouvrir la page web du Help Desk MISTRA.

Barre de menus: l'utilisateur a la possibilité de naviguer dans l'ensemble du système à l'aide de la barre de menus. Celle-ci donne accès à toutes les fonctions accessibles au profil utilisateur donné (la page de démarrage ne donne accès qu'aux plus importantes).

Choix de la langue: le choix de la langue permet à l'utilisateur de sélectionner l'allemand, le français ou l'italien.

Aide: ce bouton permet à l'utilisateur d'accéder à la page d'aide d'EMSG.

Attention: l'utilisateur n'a pas accès à toutes les fonctions dans tous les modes de travail. S'il souhaite utiliser une fonction grisée sur la page de démarrage, il doit changer de mode de travail.
