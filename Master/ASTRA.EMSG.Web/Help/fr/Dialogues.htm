EMSG Hilfe
Dialogues

La modification de données se fait généralement dans des dialogues modaux, c'est-à-dire qu'il est nécessaire de fermer la boîte de dialogue avant de pouvoir utiliser d'autres fonctions.

Il y a alors généralement quatre possibilités:

1.          Le système enregistre les données saisies. La boîte de dialogue reste ouverte, les valeurs éventuellement calculées sont mises à jour.

2.          Le système enregistre les données saisies et ferme la boîte de dialogue.

3.          Le système interrompt la modification. La boîte de dialogue est fermée sans enregistrement des données saisies.

4.          Le système supprime (après demande de confirmation) le jeu de données.

Avertissement: Les jeux de données sont physiquement effacés, c'est-à-dire qu'il n'y a pas de «corbeille» à partir de laquelle il est possible de récupérer des jeux de données supprimés par erreur. Les jeux de données dépendants (comme par exemple les informations d'état relatives à un tronçon de route) sont eux aussi perdus.
