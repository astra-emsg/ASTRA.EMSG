Log des événements

Le point de menu Log des événements (dans le menu Administration) vous permet, en tant qu'administrateur de l'application, d'avoir accès au protocole interne de l'application EMSG. Vous recevez une liste dans laquelle il est possible de faire des recherches et de réaliser des tris, et qui contient les informations suivantes:

·         Utilisateur: Identifiant de connexion de l'utilisateur ayant effectué l'action

·         Mandant: Identification de l'organisation de l'utilisateur

·         Date et heure: moment de réalisation de l'action

·         Action: action réalisée

Remarque: Les erreurs systèmes sont portées sur le fichier journal, ainsi que les actions importantes suivantes: inscription dans le système, changement du mode de travail, bilan annuel, export et import d'itinéraires d'inspection, import de fichiers Excel et prise d'un autre rôle par le support d'application.

·         Information détaillée: information supplémentaire en fonction de l'action, par exemple le nom de l'année lors d'un bilan annuel.

Le bouton Effacer le log des événements permet d'effacer la liste de manière irréversible.

Le bouton Log d'application permet de télécharger tous les fichiers du log de l'application dans un fichier ZIP à partir du serveur.

Remarque: Les fichiers logs du log de l'application contiennent bien plus d'informations techniques. En tant qu'administrateur de l'application, vous avez la possibilité de définir le degré de détails du log de l'application (voir section 7.1.1).
