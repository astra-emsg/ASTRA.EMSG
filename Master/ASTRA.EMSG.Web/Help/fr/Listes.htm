EMSG Hilfe
Listes

Des listes servent à donner une vue d'ensemble à l'utilisateur. Elles se présentent sous la forme suivante (voir Figure 3).

Figure 3: Listes dans EMSG

Les listes offrent les fonctions de base suivantes:

 Un clic sur le bouton  Ajouter un nouveau jeu de données (la légende dépend de la liste) permet d'ouvrir une boîte de dialogue. Le gestionnaire de données peut alors saisir un nouveau jeu de données.

 Un clic sur le bouton  Modifier ouvre la même boîte de dialogue pour modifier un jeu de données existant.

 Un clic sur le bouton  Supprimer supprime (après demande de confirmation) un jeu de données existant.

 Si la liste ne s'affiche pas en intégralité sur une page, les fonctions de défilement (retour à la première page, page précédente, page suivante, passage à la dernière page) permettent d'atteindre d'autres pages. Par ailleurs, il existe en bas à gauche un autre bouton permettant de réinitialiser le chargement des données (au cas où celles-ci ont été modifiées entre temps par un autre utilisateur).

Astuce: La recherche au moyen des fonctions de défilement peut s'avérer laborieuse lorsqu'il y a beaucoup de pages. Utilisez alors les fonctions de filtrage (différentes en fonction des listes) pour réduire le nombre de pages. Pour les filtrages textuels, il est possible d'utiliser le caractère «_» (tiret bas) pour filtrer les caractères de son choix.
