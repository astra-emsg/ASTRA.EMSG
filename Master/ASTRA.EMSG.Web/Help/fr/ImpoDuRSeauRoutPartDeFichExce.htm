Import du réseau routier à partir de fichiers Excel

Naviguez jusqu'à la liste de gestion des tronçons de route (voir section 4.2.1).

Un clic sur le bouton Importer le réseau routier (.xlsx) (voir Figure 11) ouvre une boîte de dialogue pour importer le réseau routier à partir d'un fichier MS Excel.

Remarque: Seul le format de fichier XLSX (Excel 2007, 2010 ou 2013) est pris en charge. Si vous utilisez une version d'Excel plus ancienne (Excel 2000, XP ou 2003), vous devez installer le «Pack de compatibilité Microsoft Office».

 

Figure 14: Boîte de dialogue «Importer le réseau routier» (sélection de fichier)

 Un clic sur le lien Télécharger le modèle ouvre un fichier Excel comprenant un formulaire d'aide pour faciliter l'import. Vous pouvez remplir le fichier et l'enregistrer en local.

 Un clic sur le bouton Sélectionner… ouvre une boîte de dialogue classique d'ouverture de fichier. Vous pouvez choisir le fichier Excel enregistré en local que vous souhaitez importer.

 Un clic sur le bouton Charger le fichier lance le contrôle du fichier Excel. Si la structure du fichier ne correspond pas au modèle ou en cas d'erreur de validation, une information détaillée s'affiche et les données ne sont pas importées.

Avec la fonction d'import, vous avez la possibilité de créer de nouveaux tronçons de route ou de mettre à jour des tronçons de route existants. Si le Nom de la route, la Désignation de et la Désignation jusqu'à d'une ligne Excel correspondent à un tronçon de route existant, ce dernier est alors mis à jour avec les informations du fichier Excel (les informations du fichier Excel viennent écraser les informations d'EMSG). Dans le cas contraire, un nouveau tronçon de route est créé.

Après contrôle, l'application affiche un aperçu des tronçons de route créés et de ceux qui ont été mis à jour (voir Figure 15). Avant de cliquer sur le bouton Exécuter import placé au-dessous de cet aperçu, prenez soin de vérifier que le résultat correspond bien à ce que vous attendiez. Toute annulation de l'import devient ensuite impossible.

Figure 15: Boîte de dialogue «Importer le réseau routier» (vérification)

Avertissement: Si, entre le chargement du fichier Excel et le démarrage de l'import, un autre utilisateur modifie l'attribut «nom de route », «désignation de» ou «désignation jusqu'à» relatifs au tronçon de route concerné, la procédure d'import ne donnera pas le résultat que vous avez visualisé sur l'aperçu. Assurez-vous qu'une telle situation ne puisse se produire au sein de votre organisation.
