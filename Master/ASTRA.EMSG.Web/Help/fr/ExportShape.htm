Export Shape

 

Figure 33: Boîte de dialogue «Export SIG»

 

Sous le point menu Export SIG vous pouvez exporter les tronçons de routes ou d‘état comme Shapefile et les importer dans d’autres systèmes SIG:

1.     Sélection de l’année de saisie (voir section 5.1.1).

2.     Sélection si tronçons de routes ou d’état doivent être exportés

3.     Aven un clic sur le bouton Télécharger, un  fichier ZIP avec le Shapefile (données géométriques et données factuelles en format Shapefile) est téléchargé.

En annexe B, vous trouvez une description des attributs utilisés dans le fichier.
