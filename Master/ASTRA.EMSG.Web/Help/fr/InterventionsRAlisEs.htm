Interventions réalisées

Le point de menu Interventions réalisées (dans le menu Interventions) donne une liste de toutes les interventions réalisées ayant été saisies. Vous avez la possibilité d'y ajouter de nouvelles interventions réalisées en saisissant le nom du projet, une description, la catégorie de charges, les coûts et les quantités (en fonction de la configuration) (voir Figure 9) et de modifier ou de supprimer des interventions réalisées existantes.

Il est possible d'effectuer une recherche par nom de projet dans la liste.

Figure 9: Boîte de dialogue «Saisir interventions réalisées» dans le mode sommaire

Remarque: Lorsque vous avez saisi de nouvelles interventions réalisées, nous vous recommandons de mettre à jour, si nécessaire, l'état moyen de votre réseau routier (voir section 4.1.1).
