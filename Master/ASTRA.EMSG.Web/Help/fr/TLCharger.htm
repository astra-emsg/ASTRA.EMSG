Télécharger

Les boutons Télécharger l'exploitation (Excel) et Télécharger l'exploitation (PDF) (voir Figure 36), ou l'outil cartographique Télécharger l'exploitation (PDF) vous permettent de télécharger une exploitation au format XLS ou PDF et de l'enregistrer en local.

Figure 36: Boutons «Télécharger l'exploitation»

Remarque: L'export sous Excel est disponible pour les graphiques et les tableaux, l'export sous PDF est disponible pour toutes les exploitations (graphiques, tableaux et cartes). Pour l'impression, les tableaux sont optimisés pour Excel et toutes les autres exploitations pour PDF.

 

Astuce: Utilisez l'export sous Excel lorsque vous souhaitez modifier les données sous Excel ou sous un autre programme. Vous avez la possibilité, par exemple, de copier un graphique facilement d'Excel vers Word ou PowerPoint.
