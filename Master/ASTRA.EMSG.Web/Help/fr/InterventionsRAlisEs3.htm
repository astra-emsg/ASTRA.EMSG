Interventions réalisées

Le point de menu Interventions réalisées (dans le menu Interventions) vous permet de saisir et de modifier des interventions réalisées avec l'éditeur de cartes.

Les interventions réalisées sont saisies de la même manière que les interventions des sous-systèmes (voir section 4.3.4). Dans la zone de dialogue (voir Figure 32), vous avez la possibilité de sélectionner plusieurs sous-systèmes concernés.

il est possible d'effectuer une recherche par nom de projet dans la liste récapitulative.

 

Figure 32: Boîte de dialogue «Modifier intervention réalisée» en mode SIG

 

Remarque: La saisie des coûts relatifs aux trottoirs est uniquement possible lorsque la largeur correspondante a également était saisie.
