Tronçons de route non encore inspectés (tableau)

Le point de sous-menu Tronçons de route non encore inspectés (tableau) vous permet, en mode noms de routes et en mode SIG, de générer une liste de tous les tronçons de route pour lesquels aucune saisie d'état n'a encore été effectuée (voir Figure 53).

Figure 53: Liste des tronçons de route non encore inspectés

Astuce: Cette exploitation peut être utilisée pour l'année de saisie actuelle afin de planifier les saisies d'état.
