EMSG Hilfe
Utilisateurs dotés du profil « gestionnaire de données »

Le gestionnaire de données utilise la fonctionnalité complète de saisie de données d'EMSG. Il traite les données relatives à son périmètre de responsabilité. Font partie du groupe des gestionnaires de données tous les utilisateurs qui saisissent les données de réseau, les données d'état ou les interventions, tels que les collaborateurs des communes ou des bureaux d'études chargés de la gestion de l'entretien pour le compte d'une commune, les collaborateurs des services cantonaux des ponts et chaussées pour les routes cantonales ou les collaborateurs de la Confédération pour les routes nationales de troisième classe.

Le gestionnaire de données peut, en fonction des besoins de son organisation (par exemple autres systèmes déjà existants), travailler dans EMSG selon trois modes différents (voir récapitulatif à la section 2.2). Le mode est défini par l'administrateur des utilisateurs pour l'ensemble de l'organisation (voir section7.2).

Si vous êtes gestionnaire de données, seul l'une des sections suivantes vous concerne, en fonction du mode configuré:

·         Mode sommaire: voir section 4.1

·         Mode nom de routes: voir section 4.2

·         Mode noms de routes: voir section 4.3
