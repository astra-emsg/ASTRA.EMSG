Mode noms de routes / Saisie détaillée avec noms de routes

Les organisations ayant décidé d'utiliser l'ensemble des fonctionnalités d'EMSG, à l'exception du SIG, ont la possibilité de décrire leur réseau routier avec des noms de route.

Les fonctions d'utilisateur suivantes sont à la disposition du gestionnaire de données en mode noms de routes:

·         Le menu Réseau routier: pour la gestion des tronçons de route avec les noms de routes (voir section4.2.3);

·         Dans le menu État, le point États et interventions proposées: pour la saisie des tronçons d'état avec les états et les interventions proposées (voir section 4.2.4);

·         Dans le menu Interventions, le point de menu Interventions réalisées: pour la saisie des interventions réalisées au cours de l'année de saisie actuelle (voir section 4.2.5);

·         Dans le menu Administration, le point de menu Valeurs caractéristiques des années précédentes: pour la saisie de valeurs caractéristiques pour les années durant lesquelles l'application métier EMSG n'était pas encore en service (identique au mode sommaire, voir section 4.1.3).
