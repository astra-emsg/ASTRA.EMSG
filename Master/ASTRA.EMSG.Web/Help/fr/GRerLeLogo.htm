Gérer le logo

Le point de menu Gérer le logo (dans le menu Administration), vous permet, en tant qu'administrateur des utilisateurs pour votre organisation, de charger le logo figurant sur toutes les exploitations (exports PDF ou Excel).

Cliquez sur le bouton Sélectionner… pour sélectionner et charger le logo à partir du système de fichiers local. Sélectionnez un fichier image au format JPEG, PNG ou BMP.

Astuce: Pour une bonne qualité d'impression, utilisez un logo haute définition. Le logo est mis à l'échelle et orienté automatiquement dans les représentations. La taille maximale de fichier est de 2 Mo.
