EMSG Hilfe
Gérer la localisation

EMSG prend en charge les langues suivantes: allemand, français et italien. L'adaptation de l'interface utilisateur d'EMSG se fait par l'intermédiaire de ce qu'on appelle les fichiers-ressources, qui sont au format XML. Le point de menu Gérer la localisation (dans le menu Administration) vous permet, en tant qu'administrateur de l'application, de télécharger ces fichiers-ressources et de charger à nouveau les fichiers téléchargés ou modifiés.

Pour adapter l'interface utilisateur, procédez de la façon suivante:

1.     Cliquez sur le bouton Télécharger les fichiers de localisation actuels pour télécharger les fichiers-ressources à partir du serveur sous la forme d'un fichier ZIP.

2.     Enregistrez le fichier ZIP en local, puis extrayez-le.

3.     Identifiez le fichier nécessaire. Les noms de fichier sont construits de la manière suivante: <contenu>Localisation.<langue>.resx, <contenu> précisant le type de textes que le fichier contient (par exemple «menu» pour toutes les entrées de menu) et <langue> correspondant à l'abréviation de la langue (de, fr, it). Les fichiers-ressources disponibles sont indiqués au
Tableau 10.

4.     Ouvrez le fichier-ressource avec un éditeur.

Astuce: Pour les petites modifications (par exemple, la correction d'une faute d'orthographe), un éditeur simple suffit (par exemple le Bloc-notes). Pour des modifications plus importantes, nous recommandons soit Microsoft Visual Studio, soit un outil spécifiquement conçu pour les fichiers-ressources (par exemple ResX Localization Studio).

 

5.     Recherchez l'emplacement concerné dans le fichier, en utilisant la fonction de recherche de l'éditeur avec la chaîne de caractères que vous souhaitez modifier.

6.     Procédez à toutes les modifications que vous souhaitez apporter, puis enregistrez le fichier.

7.     Cliquez sur le bouton Sélectionner… puis chargez le fichier-ressource modifié. Il n'est possible de charger qu'un seul fichier-ressource à la fois.

Avertissement: Toute modification des fichiers-ressources doit uniquement avoir lieu pendant la fenêtre de maintenance planifiée car, lors du chargement d'un fichier-ressource, un utilisateur en cours d'utilisation de l'application risque de perdre vos données d'entrée.

 

Remarque: Veillez à ce que les modifications apportées à l'interface utilisateur puissent également être traçables dans la documentation, surtout dans le présent Manuel de l'utilisateur (et dans le Manuel de l'utilisateur d'EMSG-Mobile [3]) ainsi que dans l'aide en ligne (voir section 7.11).
