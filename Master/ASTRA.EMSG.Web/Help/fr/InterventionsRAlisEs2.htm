Interventions réalisées

Le point de menu Interventions réalisées (dans le menu Interventions) donne une liste de toutes les interventions réalisées ayant été saisies. Vous avez la possibilité d'y enregistrer de nouvelles interventions réalisées en saisissant le nom du projet, la désignation de / jusqu'à, la longueur / la largeur, la description et les coûts de l'intervention (voir Figure 22), ainsi que de modifier ou de supprimer des interventions réalisées existantes.

Il est possible d'effectuer une recherche par nom de projet dans la liste.

Figure 22: Boîte de dialogue «Saisir interventions réalisées» en mode noms de routes

Remarque: La saisie des coûts relatifs aux trottoirs est uniquement possible lorsque la largeur correspondante a également était saisie.

 

Remarque: Lorsque vous avez saisi de nouvelles interventions réalisées, nous vous recommandons de mettre à jour les états de votre réseau routier (voir section 4.2.4).
