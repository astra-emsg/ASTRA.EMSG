Valeurs caractéristiques de l'état

Le point de menu Exploitation benchmark pour les valeurs caractéristiques de l'état (dans le menu Benchmarking) vous permet de comparer l'état de vos routes avec celui des autres communes.

Pour ce faire, il vous faut tout d'abord sélectionner l'année de référence ainsi que trois paramètres de comparaison au maximum. Cela fonctionne exactement comme pour les valeurs caractéristiques de l'inventaire (voir section 6.1) et les mêmes remarques s'appliquent.

Figure 62: Exploitation benchmark pour les « valeurs caractéristiques de l'état»

Le tableau généré (voir Figure 62) donne les valeurs caractéristiques suivantes, pondérées par les surfaces:

·         État du réseau

·         État par catégorie de charges

·         Âge des relevés d'état
