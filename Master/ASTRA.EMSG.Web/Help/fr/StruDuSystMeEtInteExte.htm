EMSG Hilfe
Structure du système et interfaces externes

L'application métier EMSG se compose des deux applications suivantes:

1.     EMSG-Master: C'est l'application principale, basée sur le Web et dotée de toutes les fonctionnalités permettant une utilisation au bureau. Cette application métier fait l'objet du présent document.

2.     EMSG-Mobile: Il s'agit d'une application sous Windows avec des fonctionnalités limitées, pouvant être exécutée sur des terminaux mobiles (tablettes PC). Elle est décrite dans [3].

L'échange de données entre EMSG-Master et EMSG-Mobile fonctionne en mode «Offline», c'est-à-dire au moyen d'un support de données (par exemple une clé USB).

Les données suivantes sont échangées:

·         De EMSG-Master vers EMSG-Mobile: Itinéraires d'inspection avec matériel cartographique associé, afin de pouvoir utiliser EMSG-Mobile sans connexion Internet. La fonction utilisateur correspondante est décrite à la section 4.3.3.

·         De EMSG-Mobile vers EMSG-Master: Données d'état recueillies et interventions proposées pour un itinéraire d'inspection. La fonction utilisateur correspondante est également décrite à la section 4.3.3.

Par ailleurs, EMSG-Master dispose des interfaces en ligne suivantes vers d'autres systèmes:

·         Système de base MISTRA 
: EMSG obtient les données axiales par l'intermédiaire du système de base, via une interface standardisée (service web ou format de fichier INTERLIS 2). La fonction utilisateur pour la mise à jour des données axiales est décrite à la section 7.5. Aucune donnée métier n'est recopiée dans le système de base.

·         LDAP: La gestion des profils d'utilisateur a lieu dans le LDAP de MISTRA. EMSG-Master lit ces informations à partir du LDAP pour attribuer des autorisations aux utilisateurs. Les profils EMSG de chaque utilisateur authentifié dans EMSG- Master sont lus à partir du service Web LDAP. Pour l'utilisateur, cela signifie une inscription automatique à EMSG avec l'identifiant MISTRA. La gestion des profils d'utilisateur se fait dans MISTRA.

·         swisstopo: Met à disposition les arrière-plans pour le mode SIG d'EMSG. Il peut s'agir d'orthophotos, de matériel cartographique ou de données MO. Les informations relatives à l'affichage des arrière-plans sont données à la section 4.3.
