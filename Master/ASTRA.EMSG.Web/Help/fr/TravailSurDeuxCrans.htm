Travail sur deux écrans

En mode SIG, lorsque des exploitations comprennent une carte et un tableau, vous avez la possibilité de les répartir sur deux fenêtres de navigateur. C'est particulièrement utile lorsque vous disposez de deux écrans. Procédez alors de la façon suivante:

1.     Dans l'onglet Liste, cliquez sur le bouton Ouvrir le tableau dans une nouvelle fenêtre. EMSG utilise alors deux fenêtres de navigateur (carte et tableau).

2.     Faites glisser l'une des deux fenêtres pour l'amener sur le second écran.

3.     Utilisez les fonctions de filtrage (voir section 5.1.1) sur les onglets. Les filtres agissent sur les deux fenêtres.

4.     Lorsque vous fermez la fenêtre du tableau, EMSG travaille de nouveau avec une seule fenêtre de navigateur, c'est-à-dire que le tableau apparaît de nouveau dans la même fenêtre que la carte.
