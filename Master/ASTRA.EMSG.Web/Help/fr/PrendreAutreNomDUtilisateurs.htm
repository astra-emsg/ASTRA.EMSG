Prendre autre (nom d')utilisateurs

Le point de menu Prendre autre (nom d')utilisateurs (dans le menu Administration) vous permet,en tant que support d'application, de prendre les droits d'un utilisateur quelconque de l'organisation (voir Figure 71).

Figure 71: Prendre un autre nom d'utilisateur

Remarque: Vous devez connaître la désignation exacte du nom d'utilisateur pour pouvoir prendre les droits d'utilisateur d'un autre utilisateur de l'organisation.
